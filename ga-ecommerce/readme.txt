== Description ==

Relying on the Google Analytics by MonsterInsights plugin, this plugin allows you to track ecommerce transactions within WooCommerce and Easy Digital Downloads.

== Installation ==

1. Make sure the Google Analytics by MonsterInsights plugin is installed, activated and configured.
2. Upload the `plugin` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Enter your license key and activate it.
5. That's it.
